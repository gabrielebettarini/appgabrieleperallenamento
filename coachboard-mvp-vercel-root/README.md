# CoachBoard MVP

MVP React/Vite per assegnare programmi palestra ai clienti.

## Deploy su Vercel

1. Crea un account GitHub.
2. Crea un nuovo repository su GitHub chiamato `coachboard-mvp`.
3. Carica tutti i file di questa cartella nel repository.
4. Vai su Vercel e scegli **Add New Project**.
5. Importa il repository GitHub.
6. Lascia i comandi automatici:
   - Build Command: `npm run build`
   - Output Directory: `dist`
7. Clicca **Deploy**.

## Nota importante

Questa è una demo frontend. Non ha ancora login, database o salvataggio permanente dei dati. Per usarla con clienti reali serve aggiungere Supabase o Firebase.
